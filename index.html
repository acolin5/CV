Airline Reservation System

This project is an airline reservation system implemented in Java, utilizing SQL to store and manage data in a database. The system allows users to perform various actions such as viewing available flights, canceling reservations, accessing their inbox, and contacting customer support. Additionally, an admin component is included, enabling administrators to verify login credentials, monitor gross sales, and generate flight log reports.

Skills Gained:

  • Java Programming: The project is primarily developed using Java, including graphical user interfaces (GUI) created with Swing.
  • SQL Database Management: SQL is used to interact with the database, including querying passenger data, canceling reservations, and fetching gross sales statistics.
  • Database Connection: The project establishes database connections and manages them to retrieve and manipulate data effectively.
  • Swing GUI Design: Graphical user interfaces (GUIs) for both users and administrators are designed using Swing components, providing a user-friendly experience.
  • JDBC (Java Database Connectivity): JDBC is utilized to connect to and interact with the MySQL database.
  • Exception Handling: The code includes error handling mechanisms to gracefully handle exceptions and display appropriate error messages to the user.

Overall, this project demonstrates proficiency in Java programming, database management, and user interface design, resulting in a functional and user-friendly airline reservation system with administrative capabilities.

TimeOn - Scheduling Application

This project is a scheduling system implemented using React Native for mobile application development and React for the web, making it accessible on both mobile devices and web browsers. The application effectively addresses the challenges faced by businesses in managing employee schedules, monitoring working hours, and optimizing resource allocation across different locations.

Skills Gained:

  • React Native: React Native was utilized to build the mobile version of the scheduling application, enabling cross-platform compatibility.
  • React for Web: React was employed to adapt the scheduling application for web browsers, ensuring a seamless user experience.
  • Firebase: Firebase integration provided real-time data storage, authentication, and database management.
  • Firebase Authentication: Firebase Authentication was used to implement user authentication and access control.
  • Firebase Hosting: Firebase Hosting ensures reliable and scalable deployment for the web version of the application.
  • React Native Libraries: Various React Native libraries were used to enhance functionality and user experience, including React Navigation and React Native Modal.

In summary, the scheduling application is a versatile solution that leverages the power of React Native, React for the web, and Firebase to provide businesses with an efficient tool for managing employee schedules and locations. Its cross-platform accessibility, real-time updates, and user-friendly interface make it a valuable asset for organizations seeking to streamline their scheduling processes and resource allocation.
